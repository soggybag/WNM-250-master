You will use the click event again here to handle clicks on the Add to Cart button.

Attributes

Attributes are the extra elements added to tags. They are always written in the form: name=value where the value is always in quotes. For example, class, href, and id are all attributes that you should be familiar with. 

<a href="#a" class="button" id="button-1">Button</a>

You can invent and add your own attributes if you like. Best practice says that non standard attribute names be prefixed with "data-". For example you might want to add the name of a product as an attribute:

data-name="Things Name"

With jQuery you can use the .attr("name") method to both get and set an attribute value.

Add to Cart

You will use this attributes for the add to cart button. You already created some JavaScript to run your shopping cart. Recall that this code had a function to add a new item to the cart When adding an item you need to tell the method the name, price, and count for the item. For example "Apple", 1.99, and 1.

When you create an add to cart button, that button will need to know which item to add, the price, and how many to add. You will make this possible by adding attributes to supply that information. The "Add to Cart" button might look like this: 

<a href="#a" class="add-to-cart" data="Apple" data-price="1.99">Add to Cart</a>

While I used an anchor tag for the example, this could be any tag type of tag. I did not include the count in this case, since clicking this button always add one Apple to the cart, I can hard code the count.

I used a class name: "add-to-cart" for this link. I did this to make it easy to work with. Anyting that adds an item to the cart will have this class. Clicking anything with this class will add an item to the cart!

Modifying the HTML of a page with jQuery

jQuery provides many ways to modify the HTML content of a page. We will explore one of these methods here: .html("")

.html("<p>Hello World</p>")

The .html() method allows you to set the HTML text within an element. For example: 

$("body").html("<h1>Hello World</h1>")

The code above sets contents of the body tag to an h1 containing the text Hello World. Note! this would replace the all other content in the body tag with the new text! Also notice the content contains tags and text. You can include classes, id's and anything else that would be valid in an html page. When using .html() imagine that you were typing the text into the page.
