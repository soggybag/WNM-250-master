CSS position

Understanding CSS position is important to being able to place things anywhere on the page in realtion to any other element. There are a few rules that govern CSS Position. Understanding these rules will make it easy for you to see where, when, and how to apply CSS position.

The position property:absolute

Absolute position allows the position of an element to be set by setting the: top, right, bottom, or left edges of the element. While this makes sense at a glance, in practice it doesn't always work out as expected. The key feature to understand with absolute position is where the coordintes are refrenced. In other words when setting the left edge of an absolutely positioned element where do start measuring from?

The answer is the realative ancestor. Normally what you will see an absolutely positioned element sets it's position relative to the edges of the browser window. This is because, if there are no other relative elements, the body tag is used as a reference.

To create define a relative ancesstor set the position of your container to realative. From here any descendant elements with the position: absolute will use the relative ancessestor as the reference.
