In this case the all of the boxes use relative position. Notice the red and blue boxes have shifted from their normal position and everything else is where it would have been if we had used the static (default) position. Setting these properties moves the object from where it would normally have been. It's relative right?

Relative position may not have an effect on object when you haven't set the left, top, right, or bottom. Relative position has a big effect on descendant objects that have absolute position, you will see the effect in a moment.
