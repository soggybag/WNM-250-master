In this example the red box is is placed on the page using absolute position. The left of the box is 100px from the left of the window, while the top of the box is 300px from the top of the window.

Other elements ignore the absolutely positioned element, and the absolute element ignores them. Absolute position is like that.

Notice the box doesn't change it's position when the window is resized, or when other elements shift around. It's position is absolute.
