Each of the four boxes uses absolute position. Notice each box ignores the other boxes. The boxes use left, top, right and bottom to set their positions. Setting the left sets the distance the left of a box is from the left edge of the window. The same is true for the top, right, and bottom.

Even though the size of the window changes the boxes are always true to their left, top, right, and bottom position. Absolute position is like that.
