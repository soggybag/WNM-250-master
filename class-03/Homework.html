Shopping Cart - Form

WNM 250 Assignment 3

The goal of this assignment is to create a shopping cart for the store you created in the previous assignment. At this stage the shopping cart will not function, it only needs to mockup it's function. We will be adding features to the project as we study JS with the goal of making the cart functional.

Your cart must have the following features

  • Shows the Name of the item.
  • An input field showing how many of each item are in the cart.
  • Show the cost of each item.
  • Show the total cost of item (cost each * number of units).
  • A button to add one unit of an item.
  • A button to delete one unit of item.
  • A button to remove all units of an item from the cart.
  • Display total number of units in the cart.
  • Display the total cost of all units in cart.

Please look at the example (shopping-cart-example-markup.html) file for an idea of how these files would be implemented. Pay close attention to the input tag and it's attributes.

  • Type - Sets the type for this input. Look into this and choose the type that is most appropriate.
  • Value - Sets the initial value.

Create your cart as a single HTML page. Style your cart with CSS. Make the cart display well at both desktop and mobile sizes.

This assignemnt should display the shoppig cart as it would work with your Store page. It shouold share the same header, footer, and other styles.

Also take a close look at the data-name attribute. This is not a standard HTML attribute. Instead this attribute was added by the the developer to include more information with these tags. While this attribute is not used here, you will be making use if it in future assignments.

Include a link to the shopping cart in your store page. Turn in the complete site, next week.
