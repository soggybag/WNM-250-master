JavaScript - Selecting elements

One of the most common tasks you will perform with JavaScript is selecting elements. Elements are arranged and held in the DOM. For the most part you can select an element similarly to haw you might select an element in CSS.

jQuery

jQuery is a library of JavaScript code that adds many helper functions to make your job of working with JavaScript easier. jQuery especially simplifies the selection of elements by adding an easy to use system that relies on the same selector syntax used by CSS.

Import jQuery

You link your HTML pages to jQuery in one of two ways.

  • Download and link to jQuery.js locally - This method works offline, but would require you to keep track of the extra file.
  • Link to jQuery on the CDN over the internet - This method requires an internet connection, but you don't have to keep track of any extra files. (Be sure to http: to the link!)

Visit jQuery.com and click Download. For the examples here I will use jQuery from the CDN. Copy the link and paste it into your page.

When does JavaScript run?

JavaScript runs as soon as the browser loads a script tag. When placing the script tag at the TOP of the page this can problems because the HTML elements lower on the page will not have loaded yet! For this reason I will ask you always place your script tags at the bottom of the page.

Any scripts that will work with jQuery require that jQuery be loaded first! Be sure to place the script that loads jQuery above any Scripts you write that will use jQuery. It's problem best practice to load jQuery before all of your other scripts.

Selecting elements

Use the $() method to select elements to work with. You can use any CSS selector string in this method. For example:

$("#close-button") // Selects the element with the id name #close-button
$("p") // Selects all p tags
$(".box") // Selects all elements with the class name .box

Adding and removing class names

A technique that I find has many practical applications is adding and removing class names. This is a simple concept that has a wide range of usefulness. You can do many things with this idea. For example hiding and showing page content. Or highlighting or showing an element as selected. Animating an element into view, are a few ideas.

The whole concept has a simple mechanism use JavaScript to add or remove a class name from an element or group of elements in your page. Connect the class name to styles in your style sheet that change the appearance of the element.

.addClass("a-class-name")

Use jQuery's addClass() method to add a class name. Include the class name as a string, do NOT prepend a dot (".")

$("#header").addClass("code-red"); // Adds the class name code-red to the element with the id #header.
$(".box").addClass("border-selected"); // Adds the class name border-selected to all elements with the class box.

.removeClass("a-class-name")

The removeClass() method removes a class name from an element. This analgous to addClass(). 

$("#header").removeClass("code-red"); // Removes the class name code-red from the element with the id #header.
$(".box").removeClass("border-selected"); // Removes the class name border-selected from all elements with the class box.

.toggleClass("a-class-name")

The toggleClass() method adds a class when that class doesn't exist, or removes the class if it does exist. Think of it as a combination of both the addClass() and removeClass() methods with some added logic.

Use toggleClass() to create things that toogle. When something toggles it changes between two states, like a light switch. You can use toggleClass() to easily create drawers that open and close, custom switches and check boxes. 

$("#header").toggleClass("code-red"); // Adds the class name code-red if it doesn't exists to the element with the id #header, otherwise it removes the class code-red.
$(".box").toggleClass("border-selected"); // Adds the class border-selected if does not exist, or removes it if it exists.

.click(function)

Use the click() method to add a click event to an object. Use this to add simple interactivity to your pages.

The click() method takes a function as a parameter, and it passes to that function an event object.

$("#header").click(function(event){
    $(this).toggleClass("code-red");
}); 
// Toggles the class code-red each time #header is clicked.
$(".box").click(function(event){
    $(this).toggleClass("border-selected"); 
});
// Toggles the class border-selected on the .box that was clicked. 
// The code here applies to all elements with the class name box. 
// The click action only applies to the single box that was clicked.

Closing the product details

For product details it would be cumbersome to write a close function for each close button. We would need one function for each detail view. It's better for us to write a generic function that applies to all of the close buttons in all of the detail views.

jQuery helps us accomplish this with some it's other methods. To solve this we need the close button to look up the DOM for the element nearest element for the element that we can identify as the container for the detail view. In my example this was: 

$(this).parents(".product-details").removeClass("show");

Here $(this) is the close button inside of the details views. Clicking it looks up the DOM with the .parents() method. In this I look at each parent trying to find an element with the class name "product-details". When I find that element I used removeClass() to remove the class "show" and the element disappears.
