Form elements

input

The input tag creates a single line input or a submit button. Yes, the input can also be a button.

Input types can be:

  • text
  • number
  • password
  • url
  • email
  • date

You can listen for a "change" at an input. When the value stored in the field changes JavaScript can respond to that change.

With jQuery you can use the .("change") to listen for changes in any form element.

To get the value from a form element with jQuery use .val()
