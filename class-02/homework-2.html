Homework 2

Due Class 3

Concept

You will create a single web page for a store. Your store must present at least 12 products. Each product must have

  • name
  • image
  • price
  • short description

You can choose any subject you like for your store. It can sell any type of product that you want to work with. Do your best to fully conceive your site. Include a logo and footer, the page should look complete. Look at some sites that you might shop at for inspiration. Ask yourself what these sites include, and why.

Your site will be responsive, it should look good on the desktop, and on your mobile phone. Columns should change size, images should scale as the browser changes width. Your site will strive to make the best use of available space.

Structure

Create a single page with a fluid design. Your finished page should look good at screen sizes for desktop, tablet and mobile devices. Pay close attention to font sizes on smaller screens. Use the inspector in Chrome to test at mobile screen sizes or test on your own mobile device!

Requirements

Use all of the ideas from class. Use only class names for your styles. Do not use id names. Use contextual tags, like section and aritcle. Do not add class names to these tags. Add class names to div, and other tags.

Use media queries to create styles that adpt for mobile. Use the ideas from class.

Write good looking well formatted code. Be sure to comment your code as you work.

Ideas from class so far

  • % width - to make elements flexible at different screen sizes
  • calc() - to make padding and margins play well with % widths
  • img 100% - To make images scale
  • @media - use media queries
    • min-width: 768px - for styles that apply to desktop and tablets
    • max-width: 767px - for styles that apply to mobile phones
    • Hide and show elements - for elements that would be hard to restyle
  • Use only class names - Do not use ID names for styles.
  • Use contextual tags - Use the best, and most descriptive markup you can think of.
