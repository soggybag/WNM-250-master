The JavaScript console

The videos above use the console to display results. The console is not displayed as a regular part of a web page. Instead the console is a tool for developers to test their work. The console can display the results from your JavaScript. Think of the console as a window into the world of JavaScript.

The command console.log("your message...") displays any message to the the developer console in the browser. You will use this to display the results of your JavaScript code.

The console also displays error messages and warnings. You will use these to troubleshoot your code. While you work you should keep the console open and look for error messages. The error message will descibe the problem. On the right side the an error will show the file where the error occured and the line number. Use this information to help track down an error.

The JavaScript console is located in different places in different browsers. Be sure you can find the console in the browser you are wokring with.

Safari

When using Safari the console, and other developer tools is not availbel in the default mode. You will need to turn on the developer options. Do this by choosing Safari > Preferences. In the preferences panel, select Advanced. This the icon on the far right. Then check the box at the bottom of the panel labeled: Show Develop menu in bar.

From here you should be able to right, or control click in the browser and choose: Inspector. Alternately you can choose Develop > Inspector.

Once the Inspector is open. The Console is located in the Tab on the right.

Chrome

In Chrome right click or control click in the browser window and choose: Inspector. Alternately choose View > Developer > JavaScript Console.

From here Choose the Console Tab.
